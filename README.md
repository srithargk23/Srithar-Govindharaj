# Tandemloop – First Screening Test  
**Candidate:** Srithar Govindharaj  
**Language Used:** JavaScript  

This repository contains my solutions for the first screening test given by Tandemloop.  
All four problems were solved using **plain JavaScript**, without using any external libraries.

---

##  Files Included

- **Problem-1.js** – Simple calculator implemented using a class  
- **Problem-2.js** – Program to generate a sequence of odd numbers up to the given input  
- **Problem-3.js** – Program to generate odd-number sequence with conditional output pattern  
- **Problem-4.js** – Program to count how many numbers are divisible by 1–9
